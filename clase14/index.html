Clase 14

Índice de Ejemplos de Clase

  • 1. Agregar botones de interacción en cada tarjeta

    Incluir botones que permitan acciones como "Me gusta", "Compartir" o "Guardar". Los botones pueden cambiar de estilo o mostrar mensajes al hacer clic, usando JavaScript para modificar el DOM.

    Prompt: Crea un ejemplo de tarjetas que incluyan botones de interacción como "Me gusta", "Compartir" y "Guardar".

  • 2. Ordenar y filtrar las tarjetas

    Implementar funciones para ordenar tarjetas por propiedades (por ejemplo, precio o fecha) y filtrar por categorías, utilizando un menú de opciones de ordenación o filtrado.

    Prompt: Implementa un sistema para ordenar y filtrar tarjetas por diferentes propiedades usando JavaScript.

  • 3. Incorporar efectos de animación o estilo en las tarjetas

    Agregar efectos de hover, sombras o transiciones CSS para hacer que las tarjetas se vean más atractivas visualmente.

    Prompt: Añade efectos de animación y estilo a las tarjetas para mejorar su presentación visual.

  • 4. Mostrar detalles adicionales al hacer clic

    Implementar un botón "Ver más" que muestre información adicional en la misma tarjeta o en un modal, para aprender a gestionar la visibilidad de contenido.

    Prompt: Crea un botón "Ver más" en las tarjetas que muestre detalles adicionales al hacer clic.

  • 5. Crear un contador o calificación dinámica en las tarjetas

    Si el JSON tiene calificaciones, mostrar estrellas o íconos en función de la calificación, o agregar un contador de "Me gusta" que se incremente al hacer clic.

    Prompt: Desarrolla un contador de "Me gusta" que se incremente al hacer clic y muestra calificaciones con estrellas.

  • 6. Implementar paginación para el listado de tarjetas

    Para muchos elementos en JSON, implementar paginación que divida las tarjetas en varias páginas o secciones, mostrando solo una cantidad específica a la vez.

    Prompt: Crea una funcionalidad de paginación para mostrar un número específico de tarjetas a la vez.

  • 7. Aplicar estilos condicionales basados en propiedades JSON

    Cambiar el color, tamaño o estilo de las tarjetas en función de una propiedad del JSON, como destacar productos "nuevos" o "en oferta".

    Prompt: Aplica estilos condicionales a las tarjetas basados en propiedades específicas del JSON.

  • 8. Incorporar una función de búsqueda en tiempo real

    Agregar un cuadro de búsqueda para filtrar las tarjetas según el texto ingresado, usando eventos de entrada y manipulación de arrays en JavaScript.

    Prompt: Implementa un cuadro de búsqueda que filtre las tarjetas en tiempo real según el texto ingresado.

  • 9. Crear grupos o secciones según categorías

    Si los datos JSON incluyen categorías, agrupar automáticamente las tarjetas en secciones, mostrando un título para cada grupo.

    Prompt: Agrupa las tarjetas en secciones basadas en categorías definidas en el JSON.

  • 10. Generar una vista en lista o cuadrícula alternable

    Crear un botón para alternar entre una vista de cuadrícula y una vista de lista, mostrando cómo alterar la presentación de los datos mediante CSS y JavaScript.

    Prompt: Desarrolla una funcionalidad que permita alternar entre vista de lista y vista de cuadrícula para las tarjetas.
